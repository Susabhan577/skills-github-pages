# GitHub Pages Blog Exercise

A hands-on GitHub Skills exercise to learn how to create and publish a blog using GitHub Pages and Jekyll.

## 📚 About This Project

This repository demonstrates how to:
- Set up GitHub Pages
- Create a Jekyll blog
- Write blog posts with proper formatting
- Deploy your site automatically

## 🚀 Getting Started

1. **Create a blog post** in the `_posts` folder
2. **Name it**: `_posts/YYYY-MM-DD-title.md`
3. **Add frontmatter** with title and date
4. **Push to main** branch
5. **View your site** at: https://Susabhan577.github.io/skills-github-pages

## 📝 Blog Post Template

```yaml
---
title: "Your Post Title"
date: 2026-06-01
---

# Your blog content here...
```
